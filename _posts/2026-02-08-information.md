整个人类已知范围由三类基本要素构成：物质，能量和信息。因而所有的行业也是围绕这三类要素提供服务：采集，加工，运输，销售。我们可以画一个二维表格概括这些行业

|      | 采集     | 加工     | 运输       | 存储     | 销售     |
| ---- | -------- | -------- | ---------- | -------- | -------- |
| 物质 | 挖矿取水 | 化工精炼 | 货运，客运 | 港口     | 百货商场 |
| 能源 | 发电站   | 电池     | 电力网络   | 电池油罐 | 加油加电 |
| 信息 | 官媒自媒 | 小说电影 | 广播电信   | 硬盘网盘 | 知识付费 |

当然，每个行业天然有顺着链条上下游延伸的动力，比如农夫山泉一家公司就解决了水的采集，加工，运输，存储和销售。另一方面，某些行业也是多个要素的叠加，例如石油的加工可以属于物质化工提取，也可以属于能源领域，送信属于物质运输也属于信息运输。因此技术的发展可能同时影响多个领域。

在中华文明几千年的历史里，信息业一直难以发展，能源的形式和使用方式也非常简陋，使得所有人都在争夺前物质要素相关的各种产业。从基础的农耕织布，到官家的贩盐铸铁。每当一个朝代兴起的周期，相关领域快速被皇亲国戚，开国元勋等人占据，然后进入垄断，最后百姓只能购买垄断企业的高价服务，不堪重负后揭竿而起。同样由于信息采集和运输的落后，执政者只有在兵临皇城后才能识别身边人编织的信息茧房。

幸运的是，随着近现代西方科学技术的发展，能源和信息领域得到了飞速的进步，能源的运输可以脱离物质的运输进行，这极大提升了效率，降低了损耗。而信息可以随着电磁波，以光速传播，并且采集手段也覆盖到视频和音频。通过机械采集而不再是口口相传或者写字作画这样经过人工参与深度加工，也极大提升了信息的可信度。所以能源和信息行业的发展带来了大量新的蛋糕，互联网的十年和AI的十年极大保证了新生代也能通过努力享受到足够的回报，信息的发展也极大提升了政府内部改革和清理的及时性，有效性，这些方面的叠加使得历史兴亡周期得以极大延展。

说回量化交易，毫无疑问整个金融业都属于信息领域，这里每天都在采集各家公司的研发，营收等信息，以及各国的政策，发展等信息。深度加工成一些所谓研报，运输到交易所，通过交易的方式将这些小众信息快速销售给市场中不知情的其他人。当然，除了直接销售给市场的交易员，还包括持续销售信息给其他人的媒体等角色。

|        | 采集                         | 加工                           | 运输           | 存储                  | 销售               |
| ------ | ---------------------------- | ------------------------------ | -------------- | --------------------- | ------------------ |
| 金融业 | 官媒自媒，公司财报，电话会议 | 政策解读，基本面分析，未来预测 | 推特，新闻联播 | Bloomberg，万德数据库 | 信息订阅，市场交易 |

因此，在市场交易本身是一种销售信息的行为（当然，绝大部分韭菜是在花钱购买别人的信息）



|          | 采集                       | 加工                 | 运输       | 存储   | 销售     |
| -------- | -------------------------- | -------------------- | ---------- | ------ | -------- |
| 量化交易 | 量价信息，基本面，另类信息 | 因子，模型，组合优化 | 光缆，微波 | 数据库 | 交易执行 |

那么，我们还可以用这套框架思考和解构量化交易。所谓量化策略，本质上也是在做信息的各环节。

对于套利类策略，只是对信息进行粗加工，最重要的是信息的采集，运输和销售。

单所套利类策略需要提升信息采集（交易所行情监控），加工（判断套利机会）和销售（下单执行）的速度，才能抢在其他人之前把信息销售给盘口未撤走的挂单。

跨所套利类策略需要重点提升信息运输的速度，因为这部分的延迟会是主要延迟。

当然，套利类策略也有信息加工的空间，比如根据微观结构对交易滑点等方面进行分析和预测。

除了套利类策略，其他预测价格类策略，比拼的主要是信息的采集（多信息类型）和加工（因子挖掘，模型调优）的能力。

通过这套框架，我们可以快速明确各类策略的核心竞争力，找到还缺乏竞争的环节获取优势。

1. 例如找到散户最多的市场去做交易，是因为必须要有足够多的买家才能卖掉信息。
2. 信息的采集要去信息的源头，获取最准确，含信息量最高的数据。
3. 信息的加工和商品加工类似，要有明确的加工目标以及质检方法。
4. 新的同类市场意味着不同的信息采集和销售的渠道，跨所策略提供了中间的信息运输服务。

还有很多量化交易的思想都可以换成商品交易来解释，同样的，从成熟的商品交易市场我们也能迁移成熟的方法论用于量化交易。

终极的量化交易范式是接入各种源头信息（白宫装摄像头？）并且能通过人物表情等信息加工到政策风向，并且找到各种潜在的关联性，然后第一时间运输到各种散户所在的市场进行售卖。现阶段靠量价变动来获取信息只能跟在内幕和新闻交易者的身后喝汤了。
